CS333 Recursion: calculator

Quick instructions:

  • Start the calculator from the Algorithm menu.
    • When prompted, give an expression to evaluate.
    • Remember to leave a blank space between each token of the expressions.

      For example, you can do "3 * ( 2 + 1 )", but not "3* (2 + 1)"

  • Use the "step" button to single-step through the code or use the "play" button to run all the way to the end of the calculation.

More detailed help is available from the Help menu.
