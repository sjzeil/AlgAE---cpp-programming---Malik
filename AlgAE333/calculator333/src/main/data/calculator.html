CS333 Recursive Calculator

Quick instructions:

  • Use the button below to launch the AlgAE demo

    or download the Jar file directly and run it on your local PC.

  • Start the calculator from the Algorithm menu.
    • When prompted, give an expression to evaluate.
    • Remember to leave a blank space between each token of the expressions.

      For example, you can do "3 * ( 2 + 1 )", but not "3* (2 + 1)"

    • >
  • When execution pauses inside of an algorithm, press the "step" button to single-step through the code or use the "play" button to resume continual execution.
  • You can drag the list nodes to new positions if you find the criss-crossing links confusing.

More detailed help is available from the Help menu.
