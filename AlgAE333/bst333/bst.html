CS333 Binary Search Trees

Quick instructions:

  • Start with an empty tree, or choose "create a random tree" from the Algorithm menu,
  • Try inserting values into the tree
  • Try searching for values in the tree (search for ones that are present and also try searchign for ones that aren't in there)
  • Starting from an empty tree, try inserting values that are already ordered: 1, 2, 3, etc.
  • Starting from an empty tree, try inserting values that are reverse-ordered: 10, 9, 8, etc.

More detailed help is available from the Help menu.
